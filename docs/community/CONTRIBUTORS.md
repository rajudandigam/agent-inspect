# Contributors

Thank you to everyone who helps improve AgentInspect.

## How to appear here

We list contributors who:

- Open merged PRs (docs, code, examples, fixtures)
- Report substantive bugs or security issues (with permission)
- Maintain issue drafts and community docs

This file is updated manually by maintainers after notable releases. It is not auto-generated from git history.

## Core maintainers

| Role | Responsibility |
| ---- | -------------- |
| Repository owner | Roadmap, releases, maintainer-owned API changes |
| Community | Triage issues, review good-first PRs, keep docs honest |
