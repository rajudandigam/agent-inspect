# v1.8.0 Execution Plan

**Train:** v1.8.0 deterministic checks, safe sharing, and CI  
**Baseline:** published v1.7.0  
**Status:** ready  
**Execution:** autonomous validated commits to `main`; one manual npm gate for first `@agent-inspect/openai-agents` publication

## Non-negotiable compatibility

- Stable root APIs and imports continue working.
- Manual/global writes remain `schemaVersion: "0.1"`.
- v0.1 and v0.2 remain readable.
- No third persisted model or schema 1.0 work.
- No root/core framework, reporter, OTel, YAML, or network dependency without an explicit stop.
- New checks/reporters remain experimental and use subpaths or optional packages.
- ESM, CJS, declarations, CLI, and Node `>=20` remain valid.
- No LLM invocation, default upload, account, collector, GitHub App, or automatic PR comment.

## Chunk protocol

Each chunk is one commit. Run focused tests while iterating, the defined chunk gate once when stable, `git diff --check`, state/task updates, diff review, fast-forward push to `main`, and required CI verification before continuing.

