## Chunk 1 — CI artifact recipe

**Goal:** Document and exemplify CI trace artifact workflows without a reporter package.

**In scope:** `docs/CI-ARTIFACTS.md`, `examples/recipes/github-actions-artifact/`, `recipes:check` registration.

**Out of scope:** `@agent-inspect/vitest-reporter`, runtime network upload, vendor sinks.

**Files:** `docs/CI-ARTIFACTS.md`, recipe folder, `scripts/validate-recipes.mjs`.

**Tests:** `pnpm recipes:check`.

**Success:** Recipe runs with `AGENT_INSPECT=1`; docs link share-safe export.

---

## Chunk 2 — `timeline` CLI

**Goal:** Chronological single-run view from v0.1 JSONL.

**In scope:** `packages/core/src/timeline.ts`, `packages/cli/src/timeline.ts`, tests.

**Out of scope:** TUI, dashboard, v0.2 default writing, network.

**CLI:** `agent-inspect timeline <runId> [--dir] [--json] [--focus slow]`

**Success:** Fixture tests pass; streaming/correlation metadata shown when present only.

---

## Chunk 3 — `stats` CLI

**Goal:** Local aggregate stats over trace directory.

**In scope:** `packages/core/src/stats.ts`, `packages/cli/src/stats.ts`, tests.

**Out of scope:** DB, cost engine, hosted analytics, mutation.

**CLI:** `agent-inspect stats [--dir] [--since] [--correlation-id] [--group-id] [--json]`

**Success:** Counts, error rate, duration percentiles, slowest runs/steps.

---

## Chunk 4 — `search` CLI

**Goal:** Deterministic local search — exact/contains matching only.

**In scope:** `packages/core/src/search.ts`, `packages/cli/src/search.ts`, tests.

**Out of scope:** Fuzzy/semantic search, index files, regex on unsafe payloads.

**CLI:** `agent-inspect search [--status] [--kind|--type] [--name] [--tool] [--duration] [--since] [--limit] [--json]`

**Success:** Deterministic ordering; no metadata dumps by default.
