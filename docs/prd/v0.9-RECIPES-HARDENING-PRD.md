## Recipes to Ship


### 1. RAG Pipeline


**File**: `examples/recipes/rag-pipeline/`


**Scenario**: Document retrieval + reranking + LLM generation


```typescript
import { inspectRun, step } from "agent-inspect";


await inspectRun("rag-pipeline", async () => {
 const query = await step("embed-query", async () => {
   return embeddings.embed("What is TypeScript?");
 });


 const docs = await step.tool("retrieve-documents", async () => {
   return vectorDB.search(query, { limit: 20 });
 });


 const reranked = await step("rerank-results", async () => {
   return reranker.rank(docs, query);
 });


 const context = reranked.slice(0, 5);


 return await step.llm("generate-answer", async () => {
   return llm.generate({ context, query });
 });
});
```


**Expected Tree**:
```text
rag-pipeline
├─ embed-query
├─ tool:retrieve-documents (20 docs)
├─ rerank-results (top 5)
└─ llm:generate-answer ✔ tokens=1.2k/450
```


### 2. Tool-Calling Failure


**File**: `examples/recipes/tool-failure-retry/`


**Scenario**: Tool fails, agent retries with different parameters


```typescript
await inspectRun("tool-retry", async () => {
 let result = null;
  for (let attempt = 1; attempt <= 3; attempt++) {
   try {
     result = await step.tool(`search-hotels-attempt-${attempt}`, async () => {
       return await searchHotels({ city: "Tokyo", flexible: attempt > 1 });
     });
     break;
   } catch (err) {
     if (attempt === 3) throw err;
     await step("adjust-params", async () => ({ flexible: true }));
   }
 }
  return result;
});
```


**Expected Tree**:
```text
tool-retry
├─ tool:search-hotels-attempt-1 ✖ error
├─ adjust-params
├─ tool:search-hotels-attempt-2 ✔ success
└─ (no attempt-3, success on 2)
```


### 3. Multi-Agent Handoff


**File**: `examples/recipes/multi-agent-handoff/`


**Scenario**: Coordinator agent delegates to specialist agents


```typescript
await inspectRun("multi-agent", async () => {
 const plan = await step("coordinator-plan", async () => {
   return { tasks: ["hotel", "flight", "car"] };
 });


 const hotelAgent = observe(new HotelAgent());
 const flightAgent = observe(new FlightAgent());


 const hotelResult = await hotelAgent.run(plan.tasks[0]);
 const flightResult = await flightAgent.run(plan.tasks[1]);


 return await step("coordinator-finalize", async () => {
   return { hotel: hotelResult, flight: flightResult };
 });
});
```


**Expected Tree**:
```text
multi-agent
├─ coordinator-plan
├─ HotelAgent.run
│  ├─ search
│  └─ select
├─ FlightAgent.run
│  ├─ search
│  └─ select
└─ coordinator-finalize
```

### 8. Retry/Fallback Pattern


**File**: `examples/recipes/retry-fallback/`


**Scenario**: Primary LLM fails, fallback to secondary


```typescript
await inspectRun("llm-fallback", async () => {
 let response;
  try {
   response = await step.llm("primary-gpt4", async () => {
     return await gpt4.generate(prompt);
   });
 } catch (err) {
   response = await step.llm("fallback-gpt35", async () => {
     return await gpt35.generate(prompt);
   });
 }
  return response;
});
```


**Expected Tree**:
```text
llm-fallback
├─ llm:primary-gpt4 ✖ error
└─ llm:fallback-gpt35 ✔ success
```


### 9. Parallel Tool Calls


**File**: `examples/recipes/parallel-tools/`


**Scenario**: Multiple tools called concurrently


```typescript
await inspectRun("parallel-search", async () => {
 const results = await Promise.all([
   step.tool("search-hotels", async () => searchHotels()),
   step.tool("search-flights", async () => searchFlights()),
   step.tool("search-cars", async () => searchCars()),
 ]);
  return await step("combine-results", async () => {
   return { hotels: results[0], flights: results[1], cars: results[2] };
 });
});
```


**Expected Tree**:
```text
parallel-search
├─ tool:search-hotels ✔ 2.3s
├─ tool:search-flights ✔ 1.8s
├─ tool:search-cars ✔ 1.2s
└─ combine-results
```


**Note**: All tools at same depth (siblings), not nested


### 10. Error Propagation


**File**: `examples/recipes/error-propagation/`


**Scenario**: Deep error in nested step


```typescript
await inspectRun("error-propagation", async () => {
 try {
   await step("outer", async () => {
     await step("middle", async () => {
       await step("inner", async () => {
         throw new Error("Database connection failed");
       });
     });
   });
 } catch (err) {
   await step("handle-error", async () => {
     return { error: err.message, recovered: true };
   });
 }
});
```


**Expected Tree**:
```text
error-propagation
├─ outer ✖ error
│  └─ middle ✖ error
│     └─ inner ✖ error ("Database connection failed")
└─ handle-error ✔ success
```

