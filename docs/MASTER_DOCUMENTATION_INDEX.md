Don't read in isolation. Open multiple documents:
- **Left monitor:** CURSOR_IMPLEMENTATION_GUIDE.md (current step)
- **Right monitor:** AGENT_INSPECT_PRD_FINAL.md (requirements)
- **Quick reference:** CURSOR_QUICK_START.md (always visible)


### Tip 2: Follow the Order


Documents are designed to be read in order:
1. README_DOCS.md (orientation)
2. CURSOR_QUICK_START.md (daily guide)
3. Step-by-step implementation
4. Advanced examples (post-MVP)

The prompts in ALL_CURSOR_PROMPTS.md are tested and complete. Don't modify unless necessary.


### Tip 4: Test After Each Step


Don't skip testing. Each step's tests catch issues early.
