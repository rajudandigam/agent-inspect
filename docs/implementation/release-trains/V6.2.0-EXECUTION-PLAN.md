# V6.2.0 — Plugin Convention (no hosted marketplace)

**Train:** v6.2.0
**Theme:** Naming-convention plugin ecosystem with explicit install and conformance.
**Authority:** [../ROADMAP_V3_5_TO_V7.md](../ROADMAP_V3_5_TO_V7.md) (§ v6.2.0)

## Release goal

Define plugin/adapter/renderer/check/importer naming conventions (`agent-inspect-plugin-*`, etc.) plus `plugins list|doctor|validate` — with explicit install only, no remote marketplace, and no untrusted code loading by default.

## Product boundary

Local/self-hosted. Explicit install only. Adapter SDK conformance + privacy checklist required. No automatic remote loading.

## Non-goals

- No hosted marketplace
- No automatic remote code loading
- No untrusted-by-default execution
