# V5.0.0 — Trace Suite Config

**Train:** v5.0.0
**Theme:** Local trajectory-evaluation layer for TypeScript agents.
**Authority:** [../ROADMAP_V3_5_TO_V7.md](../ROADMAP_V3_5_TO_V7.md) (§ v5.0.0)

## Release goal

Add `agent-inspect.suite.ts` config plus `suite` commands (`init`, `run`, `list`, `report`, `validate`) so teams define trace suites and generate readable, safe local reports.

## Product boundary

Local-first. Deterministic. No hosted dataset platform, no prompt registry, no LLM judge by default, no cloud sync.

## Non-goals

- No hosted dataset platform
- No prompt registry
- No LLM-as-judge by default
- No cloud sync / upload
