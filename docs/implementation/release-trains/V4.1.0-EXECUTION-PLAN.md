# V4.1.0 — Optional Local Index

**Train:** v4.1.0
**Theme:** Fast local search/stats/sessions/cohort over real project trace dirs.
**Authority:** [../ROADMAP_V3_5_TO_V7.md](../ROADMAP_V3_5_TO_V7.md) (§ v4.1.0)
**New package:** `@agent-inspect/index-sqlite`

## Release goal

Provide an opt-in, rebuildable, disposable local index that accelerates queries. JSONL remains source of truth; the index is never required for normal CLI use.

## Product boundary

Optional package only. No SQLite in root. No semantic/vector search in v4.1. No remote service. No background daemon by default.

## Non-goals

- No root DB dependency
- No required index for core CLI
- No vector/semantic search
- No network / upload

## Security / network impacts
