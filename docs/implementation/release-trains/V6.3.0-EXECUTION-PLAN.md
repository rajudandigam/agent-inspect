# V6.3.0 — MCP and Coding-agent Workflows

**Train:** v6.3.0
**Theme:** Read-only trace context for coding agents and local assistants.
**Authority:** [../ROADMAP_V3_5_TO_V7.md](../ROADMAP_V3_5_TO_V7.md) (§ v6.3.0)

## Release goal

Expose read-only MCP tools (read traces, search indexed traces, summarize failed run, retrieve decision notes, export share-safe bundle, find failed observation, find slowest path, run deterministic checks) with redaction enforcement, read-only by default, no code edits, no auto-fix, no replay, no hidden network.

## Product boundary

Local/self-hosted MCP server. Read-only. Redaction enforced. No unredacted fields by default. No hidden network behavior.

## Non-goals

- No code edits / auto-fix / remediation
- No replay
- No unredacted output by default
- No hidden network behavior
