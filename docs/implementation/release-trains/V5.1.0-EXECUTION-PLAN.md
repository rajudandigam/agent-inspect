# V5.1.0 — Cohort Analysis v2

**Train:** v5.1.0
**Theme:** Compare groups of runs to detect regressions.
**Authority:** [../ROADMAP_V3_5_TO_V7.md](../ROADMAP_V3_5_TO_V7.md) (§ v5.1.0)

## Release goal

Add `agent-inspect cohort` with baseline/candidate comparison, grouping (model, metadata.promptVersion, etc.), and metrics (error rate, duration, tool choice/ordering, LLM call count, token usage, retry count, observation failure, guardrail failure, circuit violation, redaction warning). Outputs `cohort-summary.md`, `cohort-results.json`, `cohort-report.html`.

## Product boundary

Local-first, deterministic ordering. No hosted analytics.

## Non-goals

- No hosted analytics / upload
- No provider cost/pricing engine
