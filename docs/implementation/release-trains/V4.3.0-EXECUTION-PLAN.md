# V4.3.0 — Shareable Trace Bundles

**Train:** v4.3.0
**Theme:** Safe, offline, PR-friendly evidence bundles.
**Authority:** [../ROADMAP_V3_5_TO_V7.md](../ROADMAP_V3_5_TO_V7.md) (§ v4.3.0)

## Release goal

Create share-safe bundles (`agent-inspect bundle ...`) with redaction and automatic `verify-safe`, defaulting to the `share` profile, failing on UNSAFE unless `--allow-unsafe`, and never mutating original traces.

## Product boundary

Local-first CLI. Redaction and verify-safe are central. Bundles are exported copies.

## Non-goals

- No upload / no hosted sharing
- No mutation of original traces
- No SaaS bundle registry
