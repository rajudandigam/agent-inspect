# V6.8.x — Adoption Feature Freeze

**Duration:** Eight to twelve weeks after v6.8.0  
**Authority:** [../ROADMAP-V6.4-TO-PRE-V7.md](../ROADMAP-V6.4-TO-PRE-V7.md) (§ v6.8.x)

## Allowed work

- Security fix
- Correctness fix
- Compatibility fix
- Packaging fix
- Upstream adapter compatibility
- Performance regression fix
- Documentation correction

## Prohibited work

- New product surface
- New persisted schema
- v7 implementation without separate authorization
- Default network behavior
- Hosted SaaS features

## Exit criteria

Update [V7.0.0-READINESS-ASSESSMENT.md](./V7.0.0-READINESS-ASSESSMENT.md) with adoption evidence. v7 decision remains conditional on manual gates.
