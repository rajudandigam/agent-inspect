# V6.0.0 — Self-hosted Studio

**Train:** v6.0.0
**Theme:** Customer-owned, self-hosted internal analyzer (no maintainer cloud).
**Authority:** [../ROADMAP_V3_5_TO_V7.md](../ROADMAP_V3_5_TO_V7.md) (§ v6.0.0)

## Release goal

Let a team run `agent-inspect studio` as a local/self-hosted read-only analyzer over multi-project workspaces (SQLite by default, optional Postgres), with localhost default and optional auth. No maintainer-operated service, no default upload.

## Product boundary

Self-hosted only. Localhost by default. Read-only routes. Explicit opt-in for any non-localhost binding or auth. JSONL/index remain source of truth.

## Non-goals

- No maintainer-hosted cloud / SaaS
- No default upload / telemetry
- No write/mutation routes
- No Postgres in root package
