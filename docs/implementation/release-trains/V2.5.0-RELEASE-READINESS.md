## Scope

v2.5.0 is the guardrails and circuit breakers train:

- `@agent-inspect/guardrails` — deterministic phrase, PII (via redact), tool-arg, injection, schema, and size rules;
- `@agent-inspect/circuit` — trace analyzers for repetition, retries, timeouts, LLM loops, branch width;
- `check --guardrails` / `check --circuit` CLI flags;
- eval factories `createEvalGuardrailRule` / `createEvalCircuitRule`;
- recipes `guardrails-basic` and `circuit-breaker-basic`.

Out of scope: remote policy engine, LLM judge, compliance certification, automatic remediation.
