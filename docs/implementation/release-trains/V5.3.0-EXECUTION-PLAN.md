# V5.3.0 — Suite Viewer

**Train:** v5.3.0
**Theme:** Make suite evidence inspectable by PM/QA/support/reviewers.
**Authority:** [../ROADMAP_V3_5_TO_V7.md](../ROADMAP_V3_5_TO_V7.md) (§ v5.3.0)

## Release goal

Add `agent-inspect viewer --suite` / `--workspace` with sections: suite list, case status, failure diff, timeline, tool path, observations, guardrails, redaction status, CI artifacts, bundle export.

## Product boundary

Local-only, read-only viewer (localhost). No upload.

## Non-goals

- No hosted viewer
- No write/mutation from viewer
- No upload
