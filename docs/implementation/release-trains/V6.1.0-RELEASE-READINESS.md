# V6.1.0 release readiness

**Train:** v6.1.0 — Client-hosted Ingestion  
**Target:** `agent-inspect@6.1.0` + linked packages @ `6.1.0` (18 packages incl. `@agent-inspect/studio`)

## Scope shipped

- RFC: [CLIENT-HOSTED-INGESTION-V6.1.md](../../proposals/CLIENT-HOSTED-INGESTION-V6.1.md)
- File-drop importer (`studio import drop`, `--ingest file-drop`)
- GitHub artifact importer (`studio import github`, operator `GITHUB_TOKEN`)
- HTTP ingest POST routes (`--ingest http`, token via `STUDIO_INGEST_TOKEN`)
- Manual bundle upload (`studio import bundle --path`)
- Idempotent `ingest_files` SQLite bookkeeping
- Security section in [SELF-HOSTING.md](../../SELF-HOSTING.md)
- Fixtures: `fixtures/studio/file-drop`, `fixtures/studio/github-artifact`

## Product boundary confirmation

- Self-hosted only; no maintainer cloud ingest
- All ingest channels **disabled by default**
- HTTP ingest requires explicit enable + token (never on by default)
- No root-package ingestion routes
- Path traversal guards on import paths
- Safe error messages; tokens never logged
- Imported content is read-only evidence; no trace mutation
