# V5.2.0 — CI Quality Gates

**Train:** v5.2.0
**Theme:** Deterministic CI gate for agent behavior.
**Authority:** [../ROADMAP_V3_5_TO_V7.md](../ROADMAP_V3_5_TO_V7.md) (§ v5.2.0)

## Release goal

Add `agent-inspect gate` (suite-driven and flag-driven: `--max-error-rate`, `--max-p95-duration`, `--forbid-tool`, `--require-observation`) with outputs `gate-results.json`, `gate-summary.md`, `gate-report.html`, `junit.xml`, `github-step-summary.md` and stable exit codes (0 pass, 1 gate failed, 2 invalid config, 3 read failure, 4 unsupported format).

## Product boundary

Local-first, CI-friendly, deterministic. No upload.

## Non-goals

- No hosted CI service
- No upload / telemetry
