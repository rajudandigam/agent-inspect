## Scope

Out of scope: hosted SaaS, marketplace, SQLite in root/core, schema breaks, IDE extension, automatic plugin loading in CLI.

## Safety

- Extension contracts are optional packages; root/core has no extension registry at startup.
- Adapter SDK defaults to metadata-only and privacy checklist guidance.
- Transform/renderer/indexer contracts are local, bounded, and rebuildable.
- Community registry is documentation only — no automatic package scanning.
