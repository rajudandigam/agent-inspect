# V5.4.0 — PM/QA Eval Templates

**Train:** v5.4.0
**Theme:** Reusable local trace-suite templates for common AI-agent product scenarios.
**Authority:** [../ROADMAP_V3_5_TO_V7.md](../ROADMAP_V3_5_TO_V7.md) (§ v5.4.0)

## Release goal

Provide `agent-inspect suite init --template <name>` templates: customer-support-agent, refund-agent, sales-assistant, browser-task-agent, MCP-tool-agent, workflow-agent, RAG-answer-agent, human-approval-agent.

## Product boundary

Local-first, deterministic templates. No API keys / real providers.

## Non-goals

- No hosted template registry
- No provider calls / upload
