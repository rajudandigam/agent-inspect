# V6.1.0 — Client-hosted Ingestion

**Train:** v6.1.0
**Theme:** Self-hosted-only ingestion of CI artifacts/bundles (no maintainer cloud).
**Authority:** [../ROADMAP_V3_5_TO_V7.md](../ROADMAP_V3_5_TO_V7.md) (§ v6.1.0)

## Release goal

Add ingestion inputs to the self-hosted Studio: HTTP ingest endpoint, file-drop directory, GitHub artifact importer, self-hosted-generated CI upload token, manual bundle upload — all disabled by default and explicitly enabled by the operator.

## Product boundary

Self-hosted only. Not in root package. Disabled by default. Explicit enable + token validation required. No public cloud maintained by AgentInspect.

## Non-goals

- No root-package ingestion
- No default-enabled endpoints
- No maintainer-operated cloud / upload target
