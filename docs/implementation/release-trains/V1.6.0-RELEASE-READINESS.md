**Status:** ready for release-preparation versioning; readiness validation passed

## Scope

v1.6.0 prepares the local runtime/writer and universal reader foundation:

- experimental `agent-inspect/writers` subpath for local writer contracts and built-in writers;
- experimental `createInspector()` / runtime isolation through `agent-inspect/advanced`;
- experimental `agent-inspect/readers` subpath with AgentInspect JSONL, OpenInference JSON, and OTLP JSON readers;
- `agent-inspect open` for local file, directory, stdin, and explicit-format ingestion;
- shared reader integration for existing local inspection commands where compatible;
- deterministic no-network recipes and documentation.

## Compatibility

- Manual writes remain `schemaVersion: "0.1"` by default.
- v0.1 and v0.2 AgentInspect JSONL traces remain readable.
- OpenInference JSON and OTLP JSON are local compatibility inputs, not a new AgentInspect persisted schema.
- Existing root imports remain valid through v1.x.
- No new root/core runtime dependency is introduced.
- Node support remains `>=20`.

## Security And Network

- No default telemetry upload, vendor sink, hosted ingestion, collector client, or replay behavior is added.
- Writer paths validate/safely reject invalid persisted events, redact/bound before disk, isolate failures, and expose stats/diagnostics.
- Reader paths are local/read-only, deterministic, warning-rich, bounded, and free of OTel/framework dependencies.
- Standards readers summarize bounded prompt/output-like fields rather than making raw prompt/output capture a default.
