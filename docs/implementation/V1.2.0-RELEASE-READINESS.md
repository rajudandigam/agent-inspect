## Scope included (v1.2.0)

- `PersistedInspectEvent` model (`schemaVersion: "0.2"`)
- `isPersistedInspectEvent` validator
- v0.1 `TraceEvent` → `PersistedInspectEvent` converters
- `InspectEvent` ↔ `PersistedInspectEvent` converters
- In-memory `PersistedInspectEvent[]` → `InspectRunTree[]` bridge (`persistedInspectEventsToRunTrees`, `traceEventsToPersistedRunTrees`)
- Legacy `TraceEvent[]` → persisted model → tree bridge
- Public exports + API stability tests
- Canonical `fixtures/traces-v0.2/` samples
- Schema/API/architecture/limitations docs + CHANGELOG unreleased section

## Scope excluded (not in v1.2.0)

- Default writing of `schemaVersion: "0.2"` files
- Storage dual-read (`readPersistedEvents` / dual-format `readTraceEvents`)
- CLI `list` / `view` / `export` / `diff` / `logs` / `tail` behavior changes
- LangChain native `0.2` persistence / streaming ([#14](https://github.com/rajudandigam/agent-inspect/issues/14))
- Timeline / stats / cohort CLI ([#11](https://github.com/rajudandigam/agent-inspect/issues/11), [#12](https://github.com/rajudandigam/agent-inspect/issues/12))
- CI reporters ([#24](https://github.com/rajudandigam/agent-inspect/issues/24))
- OTLP HTTP sink / vendor upload
- Replay engine, cost engine, SaaS/dashboard
