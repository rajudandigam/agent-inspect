## Behavioral notes for release notes

- **Manual trace writing** remains `schemaVersion: "0.1"` ‚Äî no migration required for existing traces.
- **Inspection CLI** (`view`, `timeline`, `stats`, `search`, `diff`, `export`, `what`, `report`) now **reads v0.2 `PersistedInspectEvent` JSONL** via normalization to v0.1-shaped events.
- **Subpath imports** are additive; root `"."` export unchanged ‚Äî no breaking API removals.
- **`what` / `report`** are read-only; no network calls; token display uses user-supplied metadata only.
- **`list`** metadata extraction remains v0.1-oriented; v0.2-only files may show limited list metadata until a follow-up.
