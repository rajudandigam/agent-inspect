# Implementation archive

Historical reference only. Do not use as active Codex instructions. See [../README.md](../README.md) and [../ROADMAP-V1.8.1-TO-V3.md](../ROADMAP-V1.8.1-TO-V3.md).

This directory documents the archive policy for superseded AgentInspect implementation materials. Some historical files remain in `docs/implementation/` to preserve existing links; those files should carry a historical banner when they are no longer active.

Historical or superseded examples include:

- `AGENTS-AUTONOMOUS-ADDENDUM.md`
- `CODEX-MAINTAINER-GUIDE-ADDENDUM.md`
- `README-CODEX-KIT.md`
- `PROMPT-AUTONOMOUS-V1.8.md`
- `PROMPT-RESUME-AFTER-OPENAI-FIRST-PUBLISH.md`
- `CURSOR-MAINTAINER-ROADMAP.md`
- old release-readiness records and completed release-train plans
- old roadmap execution files superseded by `ROADMAP-V1.8.1-TO-V3.md`

Do not delete historical release-readiness records. If a file is moved here later, update all links or leave a stub in the original location.
