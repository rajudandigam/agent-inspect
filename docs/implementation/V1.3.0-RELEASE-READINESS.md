## Suggested release notes draft

**Title:** agent-inspect v1.3.0 — correlation, share-safe exports, and streaming metadata

**Highlights:**

- **Correlation metadata:** optional `correlationId`, `requestId`, `decisionId`, and `groupId` on `inspectRun` / `maybeInspectRun`; `getCurrentCorrelationMetadata()` for adapters.
- **Share-safe exports:** `local` / `share` / `strict` redaction profiles on trace writing and `agent-inspect export --redaction-profile`.
- **LangChain streaming metadata:** opt-in `stream: true` records chunk counts, timing, and duration — **no full token capture by default** (`capture: "metadata-only"`).
- **Local-first boundary unchanged:** no vendor upload, no hosted dashboard, no default `schemaVersion: "0.2"` writing.

## Rollback notes
