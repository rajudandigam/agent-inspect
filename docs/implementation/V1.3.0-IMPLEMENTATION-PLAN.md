# v1.3.0 Implementation Plan — Correlation, Share-Safe Exports, Streaming Metadata

**Status:** In progress (chunk 1 — correlation metadata foundation)  
**Companion:** [V1.3.0-RELEASE-TRAIN.md](./V1.3.0-RELEASE-TRAIN.md) · [ROADMAP.md](../../ROADMAP.md)

---

## 1. Theme

Trace **correlation** + **share-safe artifacts** + **LangChain streaming metadata**.

---

## 2. Why this release exists

v1.2.0 added a **source-agnostic persisted-event foundation** (`PersistedInspectEvent`, converters, in-memory tree bridge) without changing default trace writing.

v1.3.0 makes traces easier to **group**, **share safely**, and **inspect during streaming workflows**:

- **Correlation metadata** links runs to requests, eval cases, CI jobs, and decision branches.
- **Redaction profiles** (chunk 2) tighten what leaves the machine on export.
- **LangChain streaming metadata** (chunk 3) records stream lifecycle without raw token capture by default.

Correlation supports future **CI artifacts**, **trace-to-eval**, **stats/cohort**, and multi-service debugging — all local-first.

---

## 3. Goals

- Add `correlationId` / `requestId` / `decisionId` / `groupId` foundation on `inspectRun` / `maybeInspectRun`.
- Add redaction profiles and share-safe export mode in **chunk 2**.
- Add LangChain streaming metadata-only support in **chunk 3**.
- Preserve **local-first / no-vendor-upload** boundary.
- Keep manual trace writing at **`schemaVersion: "0.1"`**.
- Keep **v0.2** as experimental foundation (converters only; not default write format).

---

## 4. Non-goals

- No `timeline` / `stats` / cohort CLI in v1.3.0
- No CI reporters in v1.3.0
- No NestJS package
- No Vercel AI SDK adapter
- No OTLP HTTP sink
- No default v0.2 writing
- No SaaS / dashboard
- No raw chain-of-thought capture

---

## 5. Chunks

### Chunk 1 — Correlation metadata foundation

- `InspectRunOptions` correlation fields → `run_started.metadata`
- `getCurrentCorrelationMetadata()` inside active runs
- Persisted converter maps correlation fields to top-level `attributes.*`
- Tests + API/docs

### Chunk 2 — Redaction profiles and share-safe export mode

- Named redaction presets beyond default keys
- Export path applies stricter profiles (opt-in)
- Docs + tests

### Chunk 3 — LangChain streaming metadata support

- Design RFC ([#14](https://github.com/rajudandigam/agent-inspect/issues/14))
- Metadata-only streaming callbacks; size bounds
- No prompt/token capture by default
